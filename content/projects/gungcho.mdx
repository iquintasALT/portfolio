## Introduction

Gungcho is a fast-paced platformer designed for LED matrix displays, where speed and precision are key.

## Features

- Platforming gameplay
- Real-time rendering on LED matrix
- Unique visual style

## Hardware Used

- Raspberry Pi 4 Model B
- 64x64 RGB LED Matrix Panel

## How it Works

Gungcho streams game frames to the LED matrix using a Node.js backend and the `node-pixel-pusher` library.
