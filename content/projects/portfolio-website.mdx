## Introduction

This portfolio website showcases modern React best practices and a beautiful, interactive UI, inspired by creative LED matrix projects.

## Features

- Responsive design
- Interactive project gallery
- Real-time previews

## Technologies Used

- React
- TypeScript
- Tailwind CSS

## How it Works

The site is built with React and TypeScript, using a modular component structure and Redux for state management. Animations and effects are inspired by LED matrix visuals.
