## Introduction

Triturados Engine is a high-performance engine for pixel art games, optimized for LED matrix displays and real-time effects.

## Features

- Pixel-perfect rendering
- Lua scripting support
- Real-time LED output

## Hardware Used

- Raspberry Pi 4 Model B
- RGB LED Matrix Panel

## How it Works

The engine streams game frames to the LED matrix using a custom Node.js backend and the `node-pixel-pusher` library.
